2. Setup redis in .env

CACHE_STORE=redis
REDIS_CLIENT=phpredis

Setup host and port, in Laravel use Cache::remember, also there are more methods to use for Cache such as get(), put(), forget().

3. Avoid select *, avoid N+1, use with() instead of loading data inside a loop. Also in .env we can use DEBUGBAR_ENABLED or QUERY_LOGGING to monitor the query amount.

5. Laravel service container used to resolving classes and managing dependencies. Reusable function or query implement in service and called from controller instead of reimplement the same function in every controller. This makes the code more easier to maintain.
